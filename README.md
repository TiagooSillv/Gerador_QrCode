# Gerador de QR Code em Node.js

Este é um projeto de gerador de QR code simples em Node.js que permite criar QR codes a partir de URLs ou qualquer outra informação desejada. É uma ferramenta útil para personalizar QR codes e compartilhar informações rapidamente.

## Como Usar

1. Clone este repositório para o seu computador:

2. Instale as dependências:
npm install

4. Acesse o gerador de QR code no seu navegador:
http://localhost:3000

5. Insira a informação desejada e clique em "Gerar QR Code" para criar o seu QR code personalizado.

## Tecnologias Utilizadas

- Node.js
- Express.js
- qr-image (biblioteca de geração de QR codes)
